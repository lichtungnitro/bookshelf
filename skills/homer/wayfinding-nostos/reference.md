# Wayfinding — reference

## Manuscript note (`src/the_odyssey_by_homer.pdf`)

The copy in this repository is a **35-page image-only PDF** (no text layer). Automated extraction yields empty strings; OCR was not available in the build environment. This skill is grounded in the **Odyssey** as a stable literary work, not in a line-level concordance to that scan.

## Episode anchors (for memory and teaching)

| Theme | Books (common division) | Brief anchor |
|-------|-------------------------|--------------|
| Lotus-eaters | ~9 | Pleasure that erases memory of home |
| Cyclops | ~9 | Hubris after victory; naming and blinding |
| Circe / underworld | ~10–11 | Delay, counsel of the dead |
| Sirens / Scylla-Charybdis | ~12 | Seduction of knowledge; forced loss |
| Phaeacians | ~6–8, 13 | Safe passage when *xenia* is honored |
| Return to Ithaca | ~13–24 | Disguise, testing, proof, restoration |

Use these as **shared vocabulary** in planning, not as proof-texting.
