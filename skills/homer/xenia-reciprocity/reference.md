# Xenia reciprocity — reference

## Manuscript note (`src/the_odyssey_by_homer.pdf`)

The repository PDF is **image-only** (no extractable text layer). This skill uses **xenia** as depicted in the **Odyssey**, not line-level extraction from that scan.

## Episode anchors

| Scene | Books (typical) | Idea |
|-------|-----------------|------|
| Telemachus and Athena/Mentes | ~1 | A guest is honored; information flows |
| Nestor, Menelaus | ~3–4 | Proper feasting, storytelling as gift |
| Cyclops | ~9 | *Xenia* violated from the host side (cannibal "hospitality") |
| Circe, Calypso | ~10, 5 | Detention dressed as care |
| Phaeacians | ~6–8, 13 | Gifts and safe passage when norms hold |
| Suitors in Odysseus' hall | ~17–22 | Parasites who invert guest and host |

## Modern mapping (loose)

| Poetic role | Often maps to |
|-------------|----------------|
| Host | Platform owner, on-call, maintainer, team that "owns" the system |
| Guest | Integrator, vendor, internal consumer, new hire |
| Gift | Code review, incident writeup, doc fix, revenue, clear requirements |

Mappings are **analogies**; do not treat coworkers as ancients.
